# JavaScript Learning 📚

Welcome to my JavaScript Learning repository! This repository contains a comprehensive collection of JavaScript fundamentals, ES6+ features, and advanced concepts with 52 practice files covering essential topics and examples.

## 📋 About This Repository

This is a structured learning journey through JavaScript, starting with the absolute basics and progressing to intermediate and advanced concepts. Each file focuses on a specific topic with practical examples and exercises.

## 🎯 Topics Covered

### 1. **Fundamentals** (Files 1-9)
- **Comments** - Writing and using comments in code
- **Variables & Data Types** - var, let, const, strings, numbers, booleans, objects
- **Operators** - Arithmetic, comparison, logical, increment/decrement operators
- **String Operations** - Concatenation, escaping literals, string indexing, template literals
- **Arrays & Nested Arrays** - Array basics, nested structures, multi-dimensional arrays
- **Array Manipulation** - push, pop, shift, unshift operations

### 2. **Functions & Scope** (Files 10-14)
- **Function Reusability** - Writing reusable functions
- **Passing Arguments** - How to pass and handle function parameters
- **Global Variables** - Understanding global scope
- **Local vs Global Variables** - Variable scope comparison and best practices
- **Queue Operations** - Practical examples with array queue operations

### 3. **Control Flow** (Files 15-18)
- **If/Else Statements** - Conditional branching
- **Strict Equality Operator (===)** - Type-safe comparisons vs loose equality
- **Logical Operators** - AND (&&), OR (||), NOT (!) operations
- **Switch Case** - Multi-way branching and switch statements

### 4. **Objects & Collections** (Files 19-24)
- **Objects** - Creating and using JavaScript objects
- **Object Lookup** - Accessing object properties
- **Testing Object Properties** - hasOwnProperty and property validation
- **Manipulating Complex Objects** - Nested objects and complex data structures
- **Nested Arrays** - Working with arrays inside objects
- **Mega Object Collections** - Large, realistic data structures

### 5. **Loops & Iteration** (Files 25-29)
- **Loops Basics** - for, while, do-while loops
- **For Loop Iteration** - Iteration with counters
- **Array Operations with Loops** - Manipulating array elements
- **Practical Examples** - Real-world loop usage
- **Ternary Operator** - Conditional shorthand (condition ? true : false)

### 6. **ES6+ Features** (Files 30-44)
- **var vs let** - Scope differences and when to use each
- **const Keyword** - Constant declarations and immutability
- **Prevent Object Mutation** - Object.freeze() and immutable patterns
- **Arrow Functions** - Modern function syntax and lexical 'this'
- **Default Parameters** - Function parameters with default values
- **Rest Operator (...)** - Collecting arguments into arrays
- **Spread Operator (...)** - Spreading array/object elements
- **Destructuring Assignment** - Extracting values from objects and arrays
- **Template Literals** - String interpolation with backticks
- **Simple Fields** - Object shorthand properties
- **Classes** - ES6 class syntax and object-oriented programming
- **Getters & Setters** - Property accessors in classes
- **Import & Require** - Module imports and CommonJS require()
- **Array Methods** - map, forEach, reduce, sort, etc.
- **Filter & Find** - Filtering and searching arrays

### 7. **Advanced Topics** (Files 45-52)
- **Advanced Array Methods** - Additional array manipulation techniques
- **Advanced Concepts** - Further exploration of JavaScript patterns and modern techniques

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Akhil-V04/Javascript_Leanring.git
   cd Javascript_Leanring
   ```

2. **Explore the files:**
   - Start with file 1 and progress sequentially
   - Each file builds on previous concepts
   - Open files in your code editor to read comments and examples

3. **Run the code:**
   - **In Node.js:**
     ```bash
     node "1. comments.js"
     ```
   - **In Browser:**
     - Open browser DevTools (F12)
     - Copy-paste code into console
     - Or create an HTML file and include the JavaScript

4. **Practice:**
   - Modify examples to understand concepts better
   - Write your own variations
   - Complete all 52 files for comprehensive learning
